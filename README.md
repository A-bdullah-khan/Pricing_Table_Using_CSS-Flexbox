DESCRIPTION:
A responsive pricing table webpage built using HTML and CSS Flexbox. This project displays multiple pricing plans with features and a clean layout that adjusts for different screen sizes.

FEATURES:
1) Three pricing plans 
2) Flexbox layout for alignment and spacing
3) Responsive design using media queries
4) Styled buttons for user interaction
5) Clean and centered layout
6) Custom Google Font integration

CONCEPTS USED:
1) HTML Document Structure
2) Div Elements
3) Unordered Lists
4) CSS Flexbox
5) Responsive Design 
6) CSS Styling
7) Typography and Fonts
8) Button Styling
